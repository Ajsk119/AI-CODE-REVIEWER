# AI CODE REVIEWER
👨‍💻 **Code Reviewer (AI-Powered)**
A simple, clean, and fast AI-powered code review tool built for developers.  
Paste your code, hit **Review**, and get instant feedback—all inside a familiar, VS Code-like interface with syntax highlighting.


✨ What This Project Does:
- Lets you paste any code and review it instantly using AI.
- Highlights code syntax (JavaScript by default) for easy readability.
- Shows line numbers—just like your favorite code editor.
- Keeps everything responsive and clean with a dark-themed UI.
-  Fully scrollable editors, designed for both short snippets and longer code blocks.


🧰 Tech Stack:
- **Frontend:** React (Vite), Prism.js, Axios
- **Backend:** Node.js, Express
- **Other:** Modern CSS (custom styling)
